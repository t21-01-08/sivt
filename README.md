# sivt.
Repozitář SIVT obsahuje 2 složky a 3 textové dokumenty:

dokument topics obsahuje názvy 5 témat, které jsme v semináři brali

složka projects obsahuje dokument project1, ve kterém jsou uvedeny úkoly a projekty, které jsme vytvořili

složka notes obsahuje dokument git_notes, ve kterém jsou poznámky k 5 základním příkazům z gitu.
